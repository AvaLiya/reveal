  1. $01110100011$

Use Prim’s algorithm to find a minimum spanning tree for the given weighted graph.
We start with the minimun weight edge $a,b$. The least weight edge incident to the tree constructed so far is edge $a,e$, with weight 2, so we add it to the tree. Next we add edge $d,e$, and then dege $c,d$. This completes the tree , whose total weight is 6.
